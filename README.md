## Бизнес цель

Создание монетизируемого мобильного приложения для автоматического определения видов птиц по фотографиям с помощью ML, предоставляющего пользователям мгновенный доступ к научно верифицированным справкам.

**Обоснование**: Существует растущая аудитория орнитологов-любителей и студентов, которые испытывают потребность в быстром и точном определении видов птиц в полевых условиях. Существующие решения либо требуют экспертных знаний, либо предоставляют недостоверную информацию. Обученная нейронная модель закрывает этот пробел, предлагая инструмент с высокой точностью распознавания, превращая наблюдение за птицами из узкоспециального хобби в доступное и технологичное занятие с монетизацией через подписку.

## Набор данных

Данные взяты из [курса по обработке изображений ИИТ](https://code.mipt.ru/courses-public/cv/public).и доступны для скачивания в публичной папке [google диска](https://drive.google.com/drive/folders/1il84dWP2z3y65jHMxXiZFAgGg8IHn18z?usp=sharing). Данные состоят из 2500 фотографий 50 видов птиц

## Метрики
1. Качество модели: точность ≥ 85%
2. Среднее время отклика сервиса на одно изображение ≤ 200 мс


## Запуск

1. Скачайте датасет из гугл-диска и положите папку в корень проекта
2. Установите все зависимости `pip install -r requirements.txt`  
3. Запустите обучение `python train.py --config config.yaml --verbose`
4. Для инференса используйте команду `python predict.py --model models/final_model/ `

**Дополнительные опции при запуске инференса**:
```bash
--test_dir <путь>      # папка с изображениями (по умолчанию из config.yaml)
--batch_size <N>       # размер батча (по умолчанию 128)
--config <путь>        # свой config.yaml
--output <файл>        # имя выходного JSON
```
